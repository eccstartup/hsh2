hsh2
====

Complement for Haskell HSH package

----
installl
```
git clone https://github.com/eccstartup/hsh2
cd hsh2
cabal configure
cabal build
cabal install
```

----
usage
```
import HSH.COmplement
ls "/"
-- list directory of root directory
ls "."
-- And anything you like
```
